# Фото для сайту

Поклади сюди 3 файли з такими **точними назвами** — і вони автоматично
з'являться на сайті (поки файлу нема, показується гарна заглушка):

| Файл | Де показується | Рекомендований формат |
|------|----------------|------------------------|
| `logo.png` | Логотип у шапці та футері | квадрат, бажано PNG з прозорим фоном |
| `about.jpg` | Блок «Про мене» (фото Володимира) | вертикальне, ~4:5 |
| `yoga.jpg` | Банер курсу з йоги в «Послугах» | горизонтальне, ~16:9 |

Можна також `.jpg`/`.png`/`.webp` — головне, щоб назва файлу збігалася
(наприклад `logo.png`). Після додавання файлів зроби commit та push —
Vercel автоматично перебудує сайт.
